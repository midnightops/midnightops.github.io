🚀 What Is Midnight Ops?

Midnight Ops is a focused collection of real-world automation tools built for engineers managing multi-vendor environments. This includes:

  • 🔗 Ansible playbooks and roles for pushing config and orchestrating changes
  • 🐍 Python scripts for REST APIs, certificate management, and automation logic
  • 🐚 Bash utilities for quick CLI-based operations and network testing
  • 📫 Postman collections for validating API integrations and automating workflows

🧰 Technologies Used

  • F5 BIG-IP Automation (certs, pools, monitors, UCS backups)
  • Cisco Meraki, ACI, SD-WAN, ISE, Firepower, and DNA Center APIs
  • Linux & shell scripting for config testing and deployment
